# Specification for UmmaLife AI (MVP)

## Project Overview
UmmaLife AI is a modular Super App for the global Muslim community. 
Tech Stack: React Native (Expo), Supabase, Claude 3.5 Sonnet API.
Design System: Luxury Modern (Deep Emerald #004D40, Matte Gold #D4AF37, Clean White).

## MVP Core Modules
1. AI Mentor / Coach (Claude 3.5 Sonnet) with "Ask without Shame" anonymous chat mode.
2. Live Halal Map with real-time restaurant menu stop-lists (Starting city: Almaty).
3. Halal-Lens (AI Product Scanner via camera).
4. Audio-Namaz Audio Guide for beginners.
